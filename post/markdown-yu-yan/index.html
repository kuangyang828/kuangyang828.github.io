标题

在Markdown中定义标题在前加#号即可。
以此类推，总共六级标题。
在#号后建议加一个空格，这是标准的Markdown语法。
# 标题H1
## 标题H3
#### 标题H4
##### 标题H5
###### 标题H6

单行代码

行内代码的语法是用反引号（Esc键）包裹。
当然单行代码也可用来标记文件名、关键字等。
示例：
这是单行代码

多行代码

方式一

用三个反引号（Esc键）包裹。
示例：

for i in range(10, 51):
    if 1 <= i % 10 <= 5:
    print(i)

方式二

缩进 4 个空格或是 1 个制表符；
注意此种方式代码块上方要留一空行
示例：

for i in range(10, 51):
    if 1 <= i % 10 <= 5:
    print(i)

列表

无序列表

无序列表以+``-``*任意开头即可
类似于标题，注意要+``-``*后面要加一个空格
示例：
+ 列表一
- 列表二
* 列表三
显示效果：

  • 列表一
  • 列表二
  • 列表三

并且可任意嵌套。

+ 一级嵌套
- 一级嵌套
* 一级嵌套
    - 二级嵌套
    * 二级嵌套
    + 二级嵌套
        + 三级嵌套
        + 三级嵌套
        + 三级嵌套
            - 四级嵌套
            + 四级嵌套
            * 四级嵌套

示例：

  • 一级嵌套
  • 一级嵌套
  • 一级嵌套
    • 二级嵌套
    • 二级嵌套
    • 二级嵌套
      • 三级嵌套
      • 三级嵌套
      • 三级嵌套

有序列表

有序列表直接以阿拉伯数学排序即可。
注意在有序列表符号后面建议也加一个空格
示例：

  1. 列表一
  2. 列表二
  3. 列表三

分割线

连续三个星号***或者连续三个横杠---即可。
推荐使用连续的三个星号。

表格

在Markdown中制作表格使用|来分隔不同的单元格，使用-来分隔表头和其他行
语法格式如下：

| 表头 | 表头 |
| --- | --- |
| 单元格 | 单元格 |

示例显示效果为：

表头 表头
单元格 单元格

我们也可以设置表格的对齐方式：

  • :---设置内容和标题左对齐
  • :---:设置内容和标题居中对齐
  • ---:设置内容和标题右对齐
    | 左对齐 | 居中对齐 | 右对齐 |
    | :--- | :---: | ---: |
    | 单元格 | 单元格 | 单元格 |
    | 单元格 | 单元格 | 单元格 |

示例显示效果为：

左对齐 居中对齐 右对齐
单元格 单元格 单元格
单元格 单元格 单元格
